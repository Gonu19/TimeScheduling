### [API 3.1] 스케줄 추천안 도출 및 제시 (GET /api/sessions/{sessionId}/recommendations)

▼ **Header**

- `Content-Type: application/json` : mandatory
    

▼ **Request**

- **Path Variable**
    
    - `sessionId` (string, UUID 형식): 추천안을 조회하고자 하는 일정 조율 방의 고유 ID
        
- **Query Parameter**
    
    - `mandatoryIds` (array of strings, optional): UI에서 필참(Mandatory)으로 지정된 멤버들의 ID 목록. 쉼표(`,`)로 구분하여 전달. (예: `?mandatoryIds=member1,member3`)
        

▼ **Response**

- Bitwise operation(비트 논리 연산)을 통해 계산된 상위 추천 스케줄 배열 (array of objects)
    
    - `rank` (integer): 추천안 순위 (1~5위)
        
    - `type` (string): 추천 알고리즘 전략 타입 ("MAX_ATTENDANCE" 또는 "MAX_CONTINUITY")
        
    - `date` (string): 추천된 일정의 날짜
        
    - `startTime` (string): 추천 시작 시간 (Format: HH:mm)
        
    - `endTime` (string): 추천 종료 시간 (Format: HH:mm)
        
    - `attendeesCount` (integer): 해당 시간에 참석 가능한 실제 멤버 수
        
    - `attendees` (array of strings): 해당 타임 슬롯에 참석 가능한 멤버 명단
        

JSON

```
[
  {
    "rank": 1,
    "type": "MAX_ATTENDANCE",
    "date": "2026-05-31",
    "startTime": "11:30",
    "endTime": "14:00",
    "attendeesCount": 4,
    "attendees": ["고은우", "강현승", "공지웅", "강호준"]
  },
  {
    "rank": 2,
    "type": "MAX_CONTINUITY",
    "date": "2026-06-02",
    "startTime": "11:30",
    "endTime": "17:30",
    "attendeesCount": 3,
    "attendees": ["고은우", "강현승", "공지웅"]
  }
]
```

▼ **Error**

- `errorCode` (string): 시스템 에러 코드
    
- `message` (string): 에러 상세 메시지
    

JSON

```
{
  "errorCode": "ERR_COMPUTATION_FAILED",
  "message": "스케줄 추천 알고리즘 연산 중 오류가 발생했습니다."
}
```

▼ **예외 (Exception Handling)**

- **세션 부재 (Not Found):** 존재하지 않는 `sessionId` 요청 시 404 Not Found 반환.
    
- **데이터 부족 (Bad Request):** 가용 시간을 제출한 멤버가 1명 이하이거나 교집합(Intersection)이 전혀 존재하지 않아 연산(Computation)이 불가능할 경우 빈 배열(`[]`)을 반환하거나 적절한 400 에러 처리.
