
### [API 1.2] 세션 정보 및 전원 가용시간 조회 (GET `/api/sessions/{sessionId}`)

▼ **Header**

- `Content-Type: application/json` : mandatory
    

▼ **Request**

- **Path Variable**
    
    - `sessionId` (string, UUID 형식): 접속 및 대시보드 렌더링을 하려는 일정 조율 방의 고유 ID
        

▼ **Response** 세션 메타 데이터 및 가용시간이 포함된 전체 멤버 명부 (object)

- `title` (string): 방 이름
    
- `domainType` (string): 팀 목적 (`"MEETING"` 또는 `"WORK"`)
    
- `status` (string): 방 상태 (`"OPEN"`, `"CONFIRMED"`)
    
- `startDate` (string): 관리자가 지정한  조율의 기준이 되는 시작 날짜
    
- `members` (array of objects): 방에 속한 전체 멤버의 Roster(명부) 및 가용시간 데이터 배열
    
    - `memberId` (long): 팀원의 고유 식별자 PK (가용시간 식별용)
        
    - `name` (string): 팀원 이름
        
    - `role` (string): 팀원 역할
        
    - `isMandatory` (boolean): 추천 알고리즘 연산 시 필수 포함 여부
        
    - `hasSubmitted` (boolean): 가용시간 최종 등록 완료 여부
        
    - `availableBitmasks` (array of long): 요일별 가용시간을 압축한 64-bit 정수 배열 (길이 7). 아직 가용시간을 제출하지 않은 경우(`hasSubmitted: false`) 모든 요소가 0으로 채워진 배열이 반환됩니다.
        

**JSON**

JSON

```
{
  "title": "소프트웨어공학 팀플 회의",
  "domainType": "MEETING",
  "status": "OPEN",
  "startDate": "2026-06-10", // 배열 대신 단일 시작 날짜 반환
  "members": [
    {
      "memberId": 1,
      "name": "강은우",
      "role": "백엔드 개발자",
      "isMandatory": true,
      "hasSubmitted": true,
      "availableBitmasks": [281474976710655, 0, 0, 0, 0, 0, 0]
    }
  ]
}
```

▼ **Error**

- `errorCode` (string): 시스템 에러 코드
    
- `message` (string): 에러 상세 메시지
    

**JSON**

JSON

```
{
  "errorCode": "ERR_SESSION_NOT_FOUND",
  "message": "존재하지 않거나 만료된 세션입니다."
}
```

▼ **예외 (Exception Handling)**

1. **세션 없음 (Not Found):** URL에 명시된 `sessionId`가 데이터베이스에 존재하지 않거나, 유효 기간이 만료되어 Purge(삭제)된 상태일 경우 **404 Not Found** 반환.
    
2. **서버 에러 (Internal Server Error):** 데이터베이스에서 멤버 엔티티 및 가용시간 비트마스크 일괄 조회 쿼리(Bulk fetch) 수행 중 인프라 결함 발생 시 **500 Internal Server Error** 반환.
